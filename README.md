# OAKHIMEDES website

Website estático do projeto OAKHIMEDES, preparado a partir dos documentos da candidatura V3.

## Estrutura

- `website/` - ficheiros públicos do site
- `website/assets/` - logotipos, imagens e ilustrações
- `versao v3/` - documentos de origem da candidatura

## Publicação

O conteúdo da pasta `website/` deve ser publicado no servidor em:

```text
http://attentia.ualg.pt/oakhimedes/
```

No servidor Nginx, a pasta de destino usada é:

```text
/var/www/oakhimedes
```
